# RentFAX Backend API

The RentFAX backend provides APIs for renter risk assessment, rental history logging, identity verification, blog publishing, Stripe payment integration, and more. It powers the RentFAX platform's core business logic.

## 🛠 Tech Stack

- **Node.js** + **Express.js**
- **MongoDB** with Mongoose
- **JWT** Auth (access + refresh)
- **Stripe** Payments
- **Persona** Identity Verification
- **AWS S3** for image storage
- **Nodemailer** for emails
